## Setup

### Cursor IDE integration

1. Open **Cursor IDE** settings
2. Navigate to `MCP` tab
3. Click on `Add new MCP server`
4. Fill in the fields:

   - **Name:** Any name
   - **Type:** `command`
   - **Command:** `npx -y @slvrio/wod`

5. Click **Save**

## Usage

> In chat (`cmd+i`), type `wod` and run the **wod tool** (`cmd+enter`)
